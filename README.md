# Woonverzekering iDIN prototype

Statische GitHub Pages-versie van het woonverzekering/iDIN prototype.

## Uploaden naar GitHub

1. Pak `woonverzekering_idin_github.zip` uit.
2. Ga naar je repository op GitHub.
3. Klik op **uploading an existing file**.
4. Sleep `index.html`, `styles.css`, `script.js`, `README.md` en de map `assets` naar GitHub.
5. Klik op **Commit changes**.

## Publiceren via GitHub Pages

1. Ga naar **Settings**.
2. Ga naar **Pages**.
3. Kies **Deploy from a branch**.
4. Branch: `main`.
5. Folder: `/ (root)`.
6. Klik **Save**.

Na 1-2 minuten staat de site op:

`https://wiljanvdlindeunive.github.io/woonverzekering_idin/`


## Variant

Deze versie volgt de flow waarbij de iDIN-check plaatsvindt in stap 4: **Uw gegevens**.
Na iDIN vult de gebruiker e-mailadres, telefoonnummer en rekeningnummer aan.
Stap 5 bevat controle van gegevens, slotvragen, marketing opt-ins en de afsluit-CTA.
